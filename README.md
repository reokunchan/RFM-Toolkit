## 概要
RFMにおけるマップ制作工程の効率化、および規格統一を目的とした制作者向け補助機能を提供します。

## 動作環境
- Minecraft: 1.21.11
- Mod Loader: Fabric
- 前提MOD: Fabric API

## 導入方法
1. Minecraft 1.21.11 に Fabric Loader をインストールしてください。
2. 本リポジトリの `Releases` より最新の jar ファイルをダウンロードしてください。
3. ダウンロードしたファイルを `mods` フォルダに配置して起動してください。
